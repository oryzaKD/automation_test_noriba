## âŒ Troubleshooting

### Token doesn't work
- Check token tidak expired
- Check permissions sudah benar (repo, workflow, admin:repo_hook)
- Regenerate token jika perlu

### Jenkins can't access repository
- Verify credentials ID is exact: `github-credentials`
- Check repository URL benar
- Test dengan verify script

### 401 Unauthorized
- Token mungkin expired
- Generate new token dengan permissions yang sama
- Update credentials di Jenkins
